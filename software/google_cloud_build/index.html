Google Cloud Build

A fully managed continuous integration, delivery, & deployment platform that lets you build, test, and deploy in the cloud. Focus on coding by running fast, consistent, reliable automated builds.

17 Alternatives To Google Cloud Build

Apache Mesos

Apache Mesos abstracts resources away from machines, enabling fault-tolerant and elastic distributed systems to easily be built and run effectively.

Azure Container Instances

Easily run application containers in the cloud with a single command. Azure Container Instances lets you get started in seconds and lower your infrastructure costs with per-second billing.

Docker Swarm

Native clustering for Docker. Turn a pool of Docker hosts into a single, virtual host.

GoCD

With its parallel and sequential execution, GoCD can easily configure dependencies for fast feedback and on-demand deployment. GoCD’s Value Stream Map lets you track a change from commit to deploy at a glance.

Marathon

Marathon is a production-grade container orchestration platform for Mesosphere.

SaltStack

SaltStack event-driven automation delivers security, cloud and configuration management for a software-defined world and enterprise complexity and scale.
