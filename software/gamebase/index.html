13 Alternatives To GameBase

CrossOver

Don’t buy a Windows license, don’t reboot and don’t use a Virtual Machine. Try a free trial of CrossOver to run your Windows software on Mac, Linux and Chrome OS.

Hataroid

Atari ST emulator for Android based on Hatari.

Javatari

Javatari is a multiplayer Atari 2600 emulator written in pure Java with no external libs.

Lakka

Lakka is the official Linux distribution of RetroArch and the libretro ecosystem.

MESS

Emulates most gaming systems that exist, emphasizing accuracy over speed.

Mame

MAME stands for Multiple Arcade Machine Emulator.

Z26

Z26 is one of the best emulators for the Atari 2600.

higan

higan was formerly known as bsnes. The project was renamed after becoming a multi-system emulator.

jGameBase
