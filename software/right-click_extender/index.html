4 Alternatives To Right-Click Extender

ConTEXT

ConTEXT is a small, fast, powerful and highly compact text editor for the Windows operating system.

MenuMaid

Free Download MenuMaid - A simple-to-use and intuitive application that helps users delete Windows Explorer and Internet Explorer context menu items, while allowing them to select the entries to be deleted

ShellMenuNew

ShellMenuNew is a small utility that displays the list of all menu items in the New submenu of Windows Explorer

ShellMenuView

ShellMenuView is a small utility that display the list of static menu items that appeared in the context menu when you right-click a file/folder on Windows Explorer
