OCRconvert is a free on-line OCR provider that enables you to convert PDF to textual content, JPEG to text and scanned images into editable files.

PDF creation with text recognition and scanner support. Features email and webpage import to PDF.

Prizmo

Prizmo is a scanning application for Mac with Optical Character Recognition (OCR) in over 40 languages with powerful editing capability, text-to-speech, and iCloud support.

Tesseract

Pure JavaScript OCR library.

WatchOCR

WatchOCR is an open source OCR server that creates searchable pdfs from images in a watched folder.

i2OCR

i2OCR is a free online Optical Character Recognition (OCR) that extracts text from images so that it can be edited, formatted, indexed, searched, or translated.
