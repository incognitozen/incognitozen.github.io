Although PyMOL is built on an open-source platform, compiling PyMOL, integrating third party tools, and learning PyMOL can be difficult.  <strong>Incentive PyMOL</strong> is a packaged software product that makes it easy to become a PyMOL user.
<ul>
 	<li>Automated Microsoft Installer for Windows</li>
 	<li>Mac OS X "drag &amp; drop"</li>
 	<li>Access to e-mail and phone support by Schrödinger Technical Support</li>
 	<li>Incentive documentation that comprehensively covers PyMOL features and updates</li>
 	<li>Narrated screencasts and step-by-step tutorials</li>
 	<li>FreeMOL, including the APBS (Adaptive Poisson-Boltzmann Solver) Plugin for PyMOL</li>
 	<li>RigiMOL, a proprietary molecular morphing software product</li>
 	<li>Open-source implementation of the MMFF94 (Merck Molecular Force Field)</li>
 	<li>MPEG encoder for simplified moviemaking</li>
 	<li>Clipboard support for copying and pasting of images in Windows and Mac</li>
 	<li>A "one window" native user interface with a Tcl/Tk/X11 option (MacPyMOL)</li>
 	<li>Direct export of QuickTime movies (Mac only)</li>
 	<li>For Schrödinger Users: PyMOL for Maestro to easily export data to PyMOL</li>
 	<li>Built-in MTZ Map Reading (and auto loading)</li>
 	<li>Stereochemical labeling (R/S enantiomers)</li>
 	<li>Improved atom typing (MOL2 and MacroModel)</li>
 	<li>Improved real-time rendering quality</li>
 	<li>Improved real-time rendering speed</li>
 	<li>Improved 'undo' command</li>
 	<li>Optimized anaglyph coloring</li>
 	<li>New mouse modes for moving lights</li>
 	<li>Ambient occlusion</li>
 	<li>Surface color smoothing</li>
 	<li>Improved feedback on the 'clean' command</li>
</ul>
