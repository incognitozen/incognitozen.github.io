Nuxt.js presets all the configuration needed to make your development of a Vue.js application enjoyable. It's a perfect static site generator.
<ul>
 	<li>Automatic transpilation and bundling (with webpack and babel)</li>
 	<li>Hot code reloading</li>
 	<li>Server-side rendering OR Single Page App OR <a href="https://hackerspad.net/classification/static-site-generators/">Static Generated</a>, you choose</li>
 	<li>Static file serving. <code>./static/</code> is mapped to <code>/</code></li>
 	<li>Configurable with a <code>nuxt.config.js</code> file</li>
 	<li>Custom layouts with the <code>layouts/</code> directory</li>
 	<li>Middleware</li>
 	<li>Code splitting for every <code>pages/</code></li>
 	<li>Loading just the critical CSS (page-level)</li>
</ul>
