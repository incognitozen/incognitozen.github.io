FreeBASIC Editor for Windows

Code is written using the FreeBASIC language and uses the Scintilla editing library (in DLL form).

The WinFBE Suite package (attached to each release as a compressed RAR file), is an all-in-one package including the latest WinFBE editor, FreeBASIC compiler (32 and 64 bit compilers), Jose Roca's WinFBX library, several Help files and all necessary Include files.

Features:
<ul>
 	<li>Unicode enabled.</li>
 	<li>32 bit and 64 bit versions.</li>
 	<li>High DPI aware for any monitor resolution.</li>
 	<li>Language localization. Simple create language file based on English.lang file.</li>
 	<li>Localized for 10 languages:</li>
 	<li>English, French, Spanish, German, Czech, Italian, Chinese, Russian, Ukrainian, Portuguese</li>
 	<li>Projects and non-project based editing.</li>
 	<li></li>
 	<li>Built-in Visual Designer for GUI application development.</li>
 	<li>Easy integration of Console or GUI based code.</li>
 	<li>Incremental compiling of code defined as a "module".</li>
 	<li>Easy including of Windows Resource files simply by specifying which source code file is the resource.</li>
 	<li>Seamless integration of the FreeBasic Compiler (both 32 bit and 64 bit).</li>
 	<li>Codetip popups and auto completion.</li>
</ul>
