Map Maker

Web

Map Maker is a powerful but easy-to-use Web application for creating custom maps. You can upload your data, customize how the data is visualized, then export the map or share it with others for editing.

Map Maker offers advanced features such as custom data importing, geocoding, data filtering, data exporting, drawing on maps and more.

  • Advanced map making
  • Easy to use
  • Geocoding
  • Map Sharing
  • Embeddable maps
  • Data analysis
